# Laundry Hero Section Task

## About This Project

This project is a simple web page for a laundry service.
It includes a navbar and a hero section.

## Features

* Navbar with logo, menu links, and username
* Hero section with text on left and image on right
* Button for booking service
* Layout made using `display: inline-block`

## Technologies Used

* HTML
* CSS

## How to Run

1. Download the files
2. Keep `index.html` and `style.css` in same folder
3. Open `index.html` in browser
