# LaundryPro - Hero Section

A clean and modern Hero section with Navbar for a laundry service web application.

## Features

- Responsive Navbar with logo and user name (no flex used)
- Eye-catching Hero section with left content and right image
- Fully viewport height design (no scroll)
- Modern design matching the sample output
- Clean, comment-free code

## Technologies Used

- HTML5
- CSS3 (Display: inline-block technique)

## How to Run

1. Download both `index.html` and `style.css`
2. Place them in the same folder
3. Open `index.html` in any modern browser
4. Deployed Link - https://newlaundry2k26.netlify.app/
