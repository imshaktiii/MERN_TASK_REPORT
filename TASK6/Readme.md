# Login Page Project

This is a simple login page created using HTML and CSS.

## What is used
- HTML for structure
- CSS for styling
- Flexbox for layout

## Features
- Full screen container (100vh)
- Two sections:
  - Left side with an image
  - Right side with login form
- Email and Password input fields
- Login button

## How to run
1. Download or copy the files
2. Keep both files in same folder:
   - index.html
   - style.css
3. Open index.html in browser
